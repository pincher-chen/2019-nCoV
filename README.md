# 2019-nCoV
A comprehensive key drug targets analysis for 2019-nCoV main protease.
### 1.吸附（adsorption）
冠状病毒突刺与受体蛋白ACE2接触.此处可以从两个方面入手，一方面针对刺突，利用刺突蛋白-ACE2得结合位点，设计新得药物阻断其结合，比如抗病毒蛋白格瑞弗森（Griffithsin）能和刺突蛋白吸附，从而达到阻断得目的；另一方面，可从宿主蛋白ACE2入手，进行靶点的药物筛选，设计CoV-ACE2的融合抑制剂，比如已筛出可能有效的沐舒坦药物。
### 2. 穿入(penertration)
病毒核酸或者感染性核衣壳穿过细胞进入胞浆
### 3. 脱壳 （uncoating）
病毒失去衣壳
### 4.生物合成 （biosynthesis）
病毒核酸复制，合成衣壳蛋白核包膜蛋白，此过程中针对CoVs复制的主要酶和蛋白质都是潜在的药物靶点。例如3CL水解酶，比如已筛出的药包括洛匹那伟，瑞得西伟。不过耐药性有待观察。

RNA依赖性RNA聚合酶（RdRp），参与病毒得转录和复制过程，参与基因组和亚基因组RNA得合成，也是可以考虑得靶点。

此外，Epro\NPro以及一些辅助蛋白不仅对病毒得组装是必须的，而且可能还具有抑制宿主免疫反应以促进病毒复制的额外功能。
### 5.装配与释放
完成病毒复制和细胞外转移过程

## 表面糖蛋白(突刺蛋白)-Surface Glycoprotein (Spike Protein, Spro)

## 3CL水解酶（主蛋白酶）-3C-like proteinase, Main Protease (Mpro)

## 包膜蛋白-Envelope protein, E Protein (Epro)

## 核衣壳蛋白-Nucleocapsid Phosphoprotein (Npro)

## 血管收缩转换酶相关的羧肽酶-Angiotensin-converting enzyme (ACE)-related carboxypeptidase, ACE2
